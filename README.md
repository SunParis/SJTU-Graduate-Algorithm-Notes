# SJTU Graduate Algorithm Course

This repository contains a compact set of algorithm notes based on the content of SJTU's graduate algorithm course taught by Prof. Li Guoqiang, together with some fast-track summary notes compiled by Codex.

## Disclaimer

These notes are intended for reference only. They are likely to be mostly correct, but they have not been formally verified line by line. If you notice anything unclear, questionable, or inconsistent, please use your own judgment and verify it independently.

## Contents

- `Note.pdf`: the exported PDF version of the notes
- `src/Note.tex`: the LaTeX source

## Intended Use

This project is meant to serve as a quick review and supplementary study resource rather than a fully rigorous textbook or official course material.
